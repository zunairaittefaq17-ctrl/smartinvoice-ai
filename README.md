# 💰 SmartInvoice AI

Professional invoice management dashboard built with Streamlit.

## Features
- Revenue analytics & metrics
- Interactive charts
- Filterable invoice tables
- CSV export
- Overdue & pending tracking

## Deploy
1. Fork this repo
2. Deploy on Streamlit Cloud
3. Done! 🚀

## Connect Your Data
Replace `load_data()` function with your database/API.
